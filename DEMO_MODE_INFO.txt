========================================

- No backend API calls
- No database needed
- Changes persist only during the session
- Refresh = data resets to default

========================================
DEMO ACTIONS YOU CAN TRY
========================================

1. View Dashboard
   - See statistics
   - View blood type inventory grid

2. Register a Donor
   - Go to Donors page
   - Click "+ Register New Donor"
   - Fill the form
   - Submit
   - See new donor in the list

3. View Inventory
   - Go to Inventory page
   - Filter by blood type (e.g., O+)
   - Filter by hospital
   - See expiring units highlighted

4. View Requests
   - Go to Requests page
   - See pending requests
   - Click "Approve" on pending request
   - Status changes to Approved
   - Click "Complete"
   - Status changes to Completed

5. View Hospitals
   - Click on any hospital
   - See hospital-specific stats
   - View inventory per hospital

6. View Reports
   - See donation statistics
   - View inventory summary
   - Analyze request patterns
